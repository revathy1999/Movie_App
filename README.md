Movie Application

A responsive React-based movie application that fetches movie data from an external API and allows users to browse, filter, and manage a personal watchlist.

🚀 Features

Fetches movie data dynamically from external API

Displays movie list with poster, title, and details

Add movies to wishlist / watch later list

Remove movies from wishlist

Filter movies based on user selection

Pagination for navigating across multiple pages

State sharing using React Context API

Component communication using React Props

🛠 Tech Stack

React

JavaScript

CSS / Tailwind CSS

Context API

External Movie API
